Etiquetas Iniciales

1. Texto básico

Para escribir un título en html, dispones de etiquetas que empiezan por "h" y son seguidas de un número, van desde h1 hasta h6, de más grande a más pequeño. En este html puedes ver que "Etiquetas básicas" es un h1 y "Títulos" un h2.

Para escribir texto normal tenemos la etiqueta "p", que es la que estoy usando para este texto y el que se encuentra arriba.

2. Listas

Las listas sirven para ordenar diferente texto, puede ser ordenada o no ordenada

  1. Primero debemos crear la etiqueta "ol", en caso de querer la lista ordena.
  2. Después, dentro de esa etiqueta, ponemos "li" y escribimos el texto dentro, esta etiqueta se usa tanto para las listas ordenadas como no ordenadas.
  1. Podemos cambiar el tipo de orden que se sigue, es decir, cómo se marca, con números o letras, por poner un ejemplo.
  2. Como puedes ver, esta lista está ordena con letras.
  3. Para hacer esto, ponemos en la etiqueta "lo" lo siguiente: type="lo que queramos". En mi caso puse "A", porque lo quería con letras, pero tenemos diferentes opciones.

  • Para las listas no ordenadas es similar.
  • La etiqueta que abre, en vez de "ol", es "ul", y ahí dentro se ponen las "li" con el texto
  • También se pueden cambiar los puntitos.
  • En este caso se haría con CSS.
  • Poniendo "list-style-type: lo que quieras;". También hay diferentes opciones, yo le puse el cuadrado, que es con "square"
  • También puedes poner cadena de texto, como se usa más adelante. Para los guines use: "- ", de esta manera la lista empieza por guión y un espacio

3. Formato de letras

Podemos cambiar el formato de letra en cualquier sitio con diferentes etiquetas. Es una etiqueta que se abre y cierra, dentro se pone el texto al que queremos aplicar el formato. Algunos ejemplos son los siguientes:

  • Para poner la letra en negrita, suamos la etiqueta "strong" o "b"
  • Para poner la letra curva, usamos la etiqueta "i"
  • Para marcar, usamos la etiqueta "mark"
  • Para subrayar, usa la etiqueta "u"
